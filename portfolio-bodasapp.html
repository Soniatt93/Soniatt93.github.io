BodasApp

Project information

  • Category: App design
  • Languages: Java, Kotlin, XML
  • Project date: February, 2021
  • Project URL: Currently private GitHub repository

Android application created from the ground up: from concept, navigation and design. Developed with Java & Kotlin in Android Studio.

This project is based on the Lean Startup methodology to develop a Minimum Viable Product using iterative cycles of design, development and testing. This application is developed for the Android mobile operating system, using as a backend a REST API created with Django REST Framework which is hosted in the cloud and taking into account the accessibility principles of Google's Material Design to design the frontend. As for the security of user data, the Django security and authentication protocol has been used.
