An in-memory web api for Angular demos and tests that emulates CRUD operations over a RESTy API generated with Angular CLI version 9.0.0. It intercepts Angular Http and HttpClient requests that would otherwise go to the remote server and redirects them to an in-memory data store. This in-memory web api service processes an HTTP request and returns an Observable of HTTP Response object in the manner of a RESTy web api.
