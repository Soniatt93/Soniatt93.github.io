Pixel Art Maker

Project information

  • Category: Web design
  • Technologies: HTML, CSS & JS
  • Project date: February, 2018
  • Project URL: Pixel Art Maker

Single-page web application that allows users to draw pixel art on a customizable canvas. This project allows the user to create a grid of squares representing their design and apply colors to those squares to create a digital masterpiece.

Project developed in Google Developer Challenge Scholarship Front End Web Dev Course.
