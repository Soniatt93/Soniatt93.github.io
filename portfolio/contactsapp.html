ContactsApp

Project information

  • Category: Web application Ionic
  • Technologies: TypeScript, HTML, SCSS, JavaScript, Firebase, Node.js
  • Tools: VSCode, GitHub Desktop
  • Project date: March, 2020
  • Project URL: ContactsApp

A simple CRUD (Create, Read, Update and Delete) using Ionic 5 to help manage Cloud Firestore data. This project was generated with Ionic CLI version 5.4.16 and Angular CLI version 8.3.25. Angular Material styles and camera plugin have also been used.
